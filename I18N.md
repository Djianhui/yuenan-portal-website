# 国际化使用文档

## 已实现功能

✅ 完整的中英文双语支持
✅ 语言切换按钮（右上角）
✅ 语言偏好本地存储
✅ 导航栏国际化
✅ 页面标题国际化
✅ 底部信息国际化

## 技术方案

使用 **Vue I18n 9.x** - Vue 3官方推荐的国际化解决方案

### 核心特性

- 🌐 支持中文（zh-CN）和英文（en-US）
- 💾 语言偏好自动保存到localStorage
- 🔄 实时切换，无需刷新页面
- 📱 响应式设计，支持移动端

## 文件结构

```
src/
├── locales/
│   ├── index.js       # i18n配置文件
│   ├── zh-CN.js       # 中文语言包
│   └── en-US.js       # 英文语言包
└── components/
    └── LanguageSwitcher.vue  # 语言切换组件
```

## 使用方法

### 1. 在组件中使用翻译

```vue
<template>
  <div>
    <h1>{{ t('common.company') }}</h1>
    <button>{{ t('common.consult') }}</button>
  </div>
</template>

<script setup>
import { useI18n } from 'vue-i18n'

const { t } = useI18n()
</script>
```

### 2. 添加新的翻译文本

在 `src/locales/zh-CN.js` 添加中文：
```javascript
export default {
  common: {
    newKey: '新文本'
  }
}
```

在 `src/locales/en-US.js` 添加英文：
```javascript
export default {
  common: {
    newKey: 'New Text'
  }
}
```

### 3. 使用翻译文本

```vue
{{ t('common.newKey') }}
```

## 语言切换

### 自动切换
用户点击右上角的语言切换按钮，自动切换语言并保存偏好

### 手动切换
```javascript
import { useI18n } from 'vue-i18n'

const { locale } = useI18n()
locale.value = 'en-US'  // 切换到英文
locale.value = 'zh-CN'  // 切换到中文
```

## 最佳实践

1. **统一命名规范**：使用小驼峰命名（camelCase）
2. **分类管理**：按页面或功能模块组织翻译文本
3. **复用文本**：公共文本放在 `common` 下
4. **保持同步**：中英文语言包结构保持一致
5. **语义化Key**：使用有意义的key名称，避免使用数字

## 测试

1. 启动开发服务器：`npm run dev`
2. 访问 http://localhost:3001
3. 点击右上角的语言切换按钮
4. 验证页面文本是否正确切换
5. 刷新页面，验证语言偏好是否保存
